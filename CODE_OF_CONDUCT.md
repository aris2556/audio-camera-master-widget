# Code Of Conduct

This project expects respectful, constructive participation.

## Expected Behavior

- Be direct without being hostile.
- Focus criticism on code, design, behavior, and documentation.
- Assume public discussion may be read by users and contributors with different experience levels.
- Respect privacy. Do not post private device IDs, logs, screenshots, or contact details without consent.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Repeated off-topic disruption after maintainers ask for it to stop.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests; or block accounts when needed to keep the project usable.

Report conduct concerns through GitHub issues only if the report can be made without exposing private information. Otherwise, ask for a private contact path.
